Android Studio安装教程(Windows版)
发表于 2019-12-09| 更新于 2019-12-13|Windows
阅读量:

1.访问 Android Studio 中文社区

2.下载Android Stuio安装包

3.下载完成运行Android Studio安装程序安装

  1. 选择下一步开始安装

​

  1. 选择是否安装安卓模拟器

​

  1. 选择Android Studio安装路径

​

  1. 选择快捷方式菜单栏的文件夹目录

​

  1. 安装完成,点击下一步

  2. 启动Android Studio

4.选择是否导入Android Studio 设置

5.设置代理

选择取消,我这里试过设置代理会出现一些问题,建议取消

6.通过安装向导继续安装

7.选择需要安装的资源

一般选择标准资源就够了

8.选择主题

9.确认设置

确认无误后点击下一步,等待下载完成,选择结束

10.管理SDK

  1. 启动Android Studio

  2. 选择Configure->SDK Manager
