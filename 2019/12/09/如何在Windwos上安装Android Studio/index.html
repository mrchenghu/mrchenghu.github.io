如何在Windwos上安装Android Studio
发表于 2019-12-09| 更新于 2019-12-17|Unity打包从入门到精通
阅读量:

访问 Android Studio 中文社区

下载Android Stuio安装包

下载完成运行Android Studio安装程序安装

  • 选择下一步开始安装

​

  • 选择是否安装安卓模拟器

​

  • 选择Android Studio安装路径

​

  • 选择快捷方式菜单栏的文件夹目录

​

  • 安装完成,点击下一步

  • 启动Android Studio

选择是否导入Android Studio 设置

设置代理

选择取消,我这里试过设置代理会出现一些问题,建议取消

通过安装向导继续安装

选择需要安装的资源

一般选择标准资源就够了

选择主题

确认设置

确认无误后点击下一步,等待下载完成,选择结束

管理SDK

  1. 启动Android Studio

  2. 选择Configure->SDK Manager
