如何在Windows上配置安卓SDK
发表于 2019-12-17| 更新于 2019-12-18|Unity打包从入门到精通
阅读量:

前言

如果你已经安装了Android Studio,Android Studio会带有SDK环境,如果没有可以通过下载ADT配置安卓SDK

访问 Android Studio 中文社区

下载ADT

ADT Bundle包含了Eclipse、ADT插件和SDK Tools，是已经集成好的IDE，只需安装好Jdk即可开始开发，推荐初学者下载ADT Bundle，不用再折腾开发环境。

  • 在主页找到下载->ADT,

  • 根据运行平台选择下载版本,我这里下载的windows64位

  • ADT下载完后是zip压缩包的格式.解压完即可使用,我的存放路径为C:\Environment\adt-bundle-windows-x86_64-20140702

可能遇到的问题

  • 打开SDK Manager.exe一闪而过无法正常打开,请检查是否正确安装了JDK并配置了Java变量

  • 打开后sdk列表无法正常加载,常见错误为dl-ssl.google.com无法连接,可尝试设置代理(以东软信息学院镜像网站为例),勾选强制https

  • 一直显示Done loading packages,无法刷新出SDK列表

    解决办法:

    1. 删除代理,取消https

    2. 手动配置hosts,打开站长工具检,在搜索栏填入 dl-ssl.google.com 选择网站测速->Ping检测

    3. 查找出ping最小的ip

    4. 打开C:\Windows\System32\drivers\etc\hosts 文件

    5. 在最下面添加你获取到的ip + ssl.google.com

    6. 关闭SDK Manager.exe重新启动,就可以正常获取到SDK了!
